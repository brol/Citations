Gestion

Vous avez simplement à àjouter une citation à la liste.
Une fois celle-ci dans la liste, elle sera disponible pour l'affichage aléatoire.
Vous pouvez aussi modifier son auteur ou corriger son contenu.
Si vous désirez supprimer une ou plusieurs citations, il vous suffit de les sélectionner et de cliquer sur le bouton Supprimer.

Intégration au thème

Vous pouvez ajouter ce code dans un fichier de votre thème pour afficher une citation aléatoire :
{{tpl:RandomCitations}}
